# Learning Objectives

At the end of this project, I should be able to explain:

## General

- What a web server is and its main role.
- What a child process is.
- Why web servers use a parent process and child processes.
- The main HTTP request methods (GET, POST, PUT, DELETE, etc.).

## DNS

- What DNS stands for (Domain Name System).
- The main role of DNS.
- The different DNS record types:
  - **A** – Maps a domain name to an IPv4 address.
  - **CNAME** – Points one domain name to another domain name.
  - **TXT** – Stores text information for a domain.
  - **MX** – Specifies the mail server responsible for receiving emails.
