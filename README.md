# Web Infrastructure * SSH & Servers

## Learning Objectives

By the end of this project, you should be able to explain the following concepts without relying on Google.

### General

* What a server is
* Where servers are typically hosted
* What SSH (Secure Shell) is
* How to create an SSH RSA key pair
* How to connect to a remote server using an SSH RSA key pair
* The advantage of using `#!/usr/bin/env bash` instead of `/bin/bash`
